# 자기소개서

## 지원 동기

클라우드 네이티브 환경에서 보안 아키텍처를 설계하고 운영하는 것은 단순히 방화벽을 설정하는 것을 넘어, 비즈니스의 민첩성과 보안성을 동시에 달성하는 균형의 예술이라고 생각합니다. 놀유니버스가 여행, 여가, 문화 플랫폼을 통합하며 수백만 사용자의 개인정보와 결제 데이터를 다루고 있다는 점에서, 클라우드 보안 아키텍처의 중요성은 아무리 강조해도 지나치지 않습니다.

저는 현재 넥스트레이드(대체거래소) 보안운영SM으로서 금융권의 엄격한 보안 요건 속에서 클라우드 인프라 보안을 설계하고 운영하고 있습니다. AWS 기반 인프라의 보안 아키텍처를 수립하고, CCE 취약점 점검을 수행하며, DevSecOps 파이프라인을 구축해온 경험을 바탕으로 놀유니버스의 안전한 클라우드 환경 구축에 기여하고 싶습니다.

## 직무 역량

**클라우드 인프라 보안 아키텍처**

넥스트레이드 프로젝트에서 AWS EKS 기반 인프라의 보안 아키텍처를 담당했습니다. VPC 네트워크 분리, Security Group 정책 설계, IAM Role 기반 최소 권한 원칙 적용 등 클라우드 네이티브 환경에서의 보안 표준을 수립했습니다. 특히 금융위원회 본인가 심사를 통과하며 규제 기관이 요구하는 수준의 보안 문서화와 아키텍처 설계 역량을 검증받았습니다.

**Kubernetes 런타임 보안**

컨테이너 오케스트레이션 환경에서의 보안은 전통적인 호스트 보안과는 다른 접근이 필요합니다. Pod Security Policy(현 Pod Security Standards), Network Policy, RBAC 등 K8s 네이티브 보안 기능을 활용한 런타임 보안 강화 경험이 있습니다. 또한 컨테이너 이미지 스캐닝과 취약점 관리를 CI/CD 파이프라인에 통합하여 빌드 시점부터 보안을 확보하는 Shift-Left 보안 전략을 적용해왔습니다.

**DevSecOps 및 자동화**

보안은 개발 프로세스의 마지막 관문이 아닌, 전 과정에 녹아들어야 한다고 믿습니다. GitLab CI/CD와 GitHub Actions를 활용해 코드 커밋부터 배포까지 보안 검증을 자동화했습니다. SonarQube(SAST), OWASP ZAP(DAST), Trivy(컨테이너 취약점), gitleaks(시크릿 스캐닝) 등을 파이프라인에 통합하여 개발 속도를 저해하지 않으면서도 보안을 확보하는 균형점을 찾았습니다.

**인프라 취약점 점검 (CCE)**

시스템 보안 설정 취약점(CCE) 점검을 정기적으로 수행하고 개선 조치를 이행해왔습니다. Ansible을 활용해 보안 설정 점검을 자동화하고, 미준수 항목에 대한 자동 교정 스크립트를 개발했습니다. 이를 통해 수동 점검 대비 70% 이상의 시간을 단축하면서도 점검 범위는 확대할 수 있었습니다.

## 협업 및 커뮤니케이션

보안 조직은 종종 "No"라고 말하는 부서로 인식됩니다. 하지만 저는 보안이 비즈니스를 가능하게 하는 Enabler가 되어야 한다고 생각합니다. 넥스트레이드에서 개발팀, 인프라팀과 협업하며 보안 요건을 전달할 때, 단순히 "이건 안 됩니다"가 아닌 "이렇게 하면 보안도 확보하면서 원하시는 기능을 구현할 수 있습니다"라는 대안을 제시해왔습니다.

놀유니버스 정보보호기술 조직이 "협업"을 가장 큰 강점으로 언급한 것에 깊이 공감합니다. 보안 표준 가이드를 일방적으로 전달하는 것이 아니라, 유관 부서와 함께 현실적인 보안 아키텍처를 설계하는 Pro-active한 커뮤니케이션이야말로 효과적인 보안의 핵심입니다.

## 성장 의지

클라우드 보안 기술은 빠르게 진화하고 있습니다. CNAPP(Cloud-Native Application Protection Platform), CSPM, CWPP 등 새로운 보안 패러다임에 대한 학습을 지속하고 있습니다. 특히 Zero Trust 아키텍처와 Policy-as-Code에 깊은 관심이 있으며, OPA(Open Policy Agent)를 활용한 정책 자동화를 개인 프로젝트에서 실험하고 있습니다.

CCNP, RHCSA, CompTIA Linux+ 등의 자격증은 네트워크와 시스템에 대한 기초 역량을, 그리고 현재 진행 중인 클라우드 보안 관련 학습은 최신 트렌드를 따라가기 위한 노력을 보여줍니다. 놀유니버스에서 클라우드 보안 아키텍처 전문가로 성장하고 싶습니다.

## 맺음말

놀유니버스의 Security Operation Manager로서 클라우드 인프라 보안 아키텍처 설계, K8s 런타임 보안 강화, DevSecOps 파이프라인 고도화에 기여하고 싶습니다. 금융권에서 쌓은 엄격한 보안 마인드와 자동화 역량을 바탕으로, 놀유니버스의 안전하고 seamless한 보안 환경 구축에 함께하고 싶습니다.
