## Q1. JYP 엔터테인먼트에 관심을 가지게 된 계기 (500자)

JYP의 AX(Artist Experience) 비전이 제가 지난 1년간 몰두해온 AI Agent 개발과 정확히 맞닿아 있어 지원하게 되었습니다.

현재 저는 구직 과정 자체를 자동화하는 AI Agent 시스템을 개발하고 있습니다. MCP(Model Context Protocol) 기반의 12개 도구를 직접 설계하고, n8n 워크플로우와 Slack 알림을 연동해 "채용공고 탐색 → 매칭 스코어링 → 자동 지원 → 결과 알림"까지 사람 개입 없이 동작하는 파이프라인을 구축했습니다. 이 경험을 통해 AI가 반복적인 업무를 대신하고, 사람은 창의적인 결정에 집중할 수 있다는 것을 체감했습니다.

AX가 추구하는 "아티스트가 음악에만 집중할 수 있는 환경"은 같은 철학입니다. 스케줄 관리, 콘텐츠 분석, 팬 인터랙션 최적화 등 아티스트를 둘러싼 운영 업무를 AI로 자동화하면, 창작자는 정말 창작에만 몰입할 수 있습니다.

기술 역량 면에서도 금융권 보안 인프라 8년 경험과 DevSecOps 자동화 노하우가 있어, 안정성과 확장성을 갖춘 시스템을 설계할 수 있습니다. JYP에서 AX 비전을 기술로 실현하고 싶습니다.

## Q2. 본인의 성격이 해당 직무에 적합한 이유 + 문제해결 사례 (1000자)

### 성격적 강점: "일단 동작하게 만들고, 반복되면 자동화한다"

저는 문제를 마주하면 완벽한 설계보다 먼저 동작하는 프로토타입을 만듭니다. 그리고 같은 작업이 두 번 이상 반복되면 반드시 자동화합니다. 이 습관이 AI/자동화 직무에 적합하다고 생각합니다.

### 문제해결 사례: 1,800석 재택전환 자동화 (메타넷, 2020)

**상황**: 코로나19로 콜센터 1,800석을 2주 내 재택근무 환경으로 전환해야 했습니다. 수동으로 VPN 계정 생성, NAC 정책 적용, 방화벽 규칙 추가를 하면 1인당 30분, 총 900시간이 필요했습니다.

**행동**: Ansible 플레이북을 작성해 전 과정을 자동화했습니다.

- VPN 계정 생성 → AD 연동 스크립트
- NAC 정책 → 부서별 템플릿 자동 적용
- 방화벽 → CSV 기반 대량 규칙 생성

3일 만에 자동화 파이프라인을 완성하고, 나머지 11일은 예외 케이스 처리와 모니터링에 집중했습니다.

**결과**:

- 예정 대비 3일 조기 완료
- 수동 작업 900시간 → 자동화 후 50시간 (94% 절감)
- 이후 동일 프로세스를 신규 입사자 온보딩에 재활용

**배운 점**: 긴급 상황에서도 "당장 손으로 하자"는 유혹을 이기고 자동화에 투자하면, 결국 더 빠르고 정확하게 끝납니다. 이 마인드셋이 AI 시스템 개발에서도 핵심이라고 생각합니다. AI가 대신할 수 있는 반복 작업을 식별하고, 사람은 예외와 창의적 판단에 집중하는 구조를 만드는 것이 제가 잘하는 일입니다.

## Q3. 지원한 직무를 수행하는 데 있어 본인이 갖춘 핵심 역량 (1000자)

### 1. AI Agent 시스템 설계 및 구현 경험

MCP(Model Context Protocol) 기반 AI Agent를 직접 설계·구현한 경험이 있습니다. 현재 운영 중인 구직 자동화 시스템에서:

- 12개 MCP 도구 (채용공고 검색, 이력서 동기화, 자동 지원 등)
- Fastify 기반 HTTP API 서버 (10개 라우트 모듈)
- 4단계 파이프라인: Search → Filter → Apply → Notify
- Slack 실시간 알림 연동

이 시스템으로 매일 새벽 자동으로 채용공고를 수집하고, 매칭 스코어 기반으로 필터링한 뒤, 조건에 맞으면 자동 지원까지 수행합니다.

### 2. Workflow 자동화 (n8n, Ansible)

복잡한 비즈니스 로직을 시각적 워크플로우로 구현하는 데 능숙합니다:

- n8n: AI Agent 트리거, 슬랙 알림, 스케줄링
- Ansible: 인프라 프로비저닝, 보안정책 자동 적용
- 1,800석 재택전환 사례에서 94% 시간 절감 달성

### 3. 빠른 프로토타이핑과 배포

아이디어에서 동작하는 시스템까지 빠르게 만듭니다:

- Cloudflare Workers 기반 포트폴리오 사이트 (1MB 단일 번들)
- Go CLI로 REST API 배포 자동화
- Jest 90% 커버리지 + Playwright E2E 테스트

### 4. 금융권 보안 인프라 8년 경험

민감한 데이터를 다루는 시스템에서 보안과 안정성을 동시에 확보하는 방법을 압니다:

- 금감원 본인가 심사 대응 (NEXTRADE)
- 로보어드바이저 테스트베드 심사 (콴텍)
- 망분리 환경에서의 시스템 운영 (KAI, 콴텍)

AX 시스템도 아티스트의 민감한 정보를 다루므로, 이 경험이 직접적으로 도움이 됩니다.

## Q4. 엔터테인먼트 산업에 대한 본인의 생각과 미래 (500자)

엔터테인먼트 산업의 미래는 **창작의 민주화**와 **운영의 자동화**에 있다고 생각합니다.

AI 도구가 발전하면서 음악 제작, 안무 시각화, 콘텐츠 편집의 진입 장벽이 낮아지고 있습니다. 동시에 스케줄 관리, 팬 데이터 분석, 콘텐츠 성과 측정 같은 운영 업무는 점점 더 복잡해지고 있습니다. 이 간극을 AI로 메울 수 있습니다.

JYP의 AX가 정확히 이 방향을 향하고 있다고 봅니다. 아티스트가 창작에만 집중할 수 있도록 주변의 모든 것을 AI가 보조하는 환경. 예를 들어:

- **스케줄 최적화**: 컨디션, 이동시간, 휴식을 고려한 AI 스케줄링
- **팬 인터랙션 분석**: SNS 반응 패턴 분석으로 콘텐츠 전략 제안
- **연습 피드백**: 안무/보컬 연습 영상 분석 및 개선점 제시

저는 이런 시스템을 직접 설계하고 구현할 수 있는 역량을 갖추고 있습니다. AI Agent 개발 경험, 워크플로우 자동화 노하우, 보안 인프라 설계 능력을 결합해 JYP만의 AX 플랫폼을 만들고 싶습니다.
